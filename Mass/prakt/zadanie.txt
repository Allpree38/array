Задание 
Задание: Массивы и их методы 

Задание 1. Найти второй по величине элемент в массиве
Задача: Напишите функцию, которая возвращает второй по величине уникальный элемент массива чисел.

// пример: [3, 5, 7, 5, 7, 9] -> 7
Задание 2. Определить, есть ли подряд идущие одинаковые элементы
Задача: Проверить, есть ли в массиве два одинаковых элемента, следующие друг за другом.
// пример: [1, 2, 2, 3] -> true
// пример: [1, 2, 3, 4] -> false


Задание 3. Посчитать сумму элементов, расположенных на нечетных позициях
Задача: Посчитать сумму элементов массива, индексы которых — нечетные числа.

// пример: [1, 2, 3, 4, 5] -> 2 + 4 = 6


Задание 4. Проверить, является ли массив палиндромом
Задача: Проверить, одинаков ли массив если его прочитать с конца и с начала. 

// пример: [1, 2, 3, 2, 1] -> true
// пример: [1, 2, 3, 4] -> false


5. Найти все уникальные элементы массива
Задача: Вернуть массив с уникальными элементами, его элементы не должны повторяться.

// пример: [1, 2, 2, 3, 4, 4] -> [1, 2, 3, 4]


6. Разбить массив по определенному условию
Задача: Разделить массив на два  подмассива. Первый подмассив содержит только положительные числа из исходного, а второй ноль и отрицательные числа. 

// пример: [-1, -4, 7,-2, 8, 10] -> [[-1, -2, -4], [7, 8], [10]]

Задание 7. Объединить два отсортированных массива в один отсортированный
Задача: Реализовать слияние двух  массивов в один отсортированный массив. 

// например: [1, 3, 5], [2, 4, 6] -> [1, 2, 3, 4, 5, 6]


Задание 8. Проверить, есть ли в массиве хотя бы один элемент, являющийся числом, кратным 5
Задача: Вывести true, если есть хотя бы один такой элемент, иначе — false.

// пример: [1, 2, 3, 15] -> true
// пример: [1, 2, 3, 4] -> false


Задание 9. Создать массив, в котором каждый элемент — сумма текущего и следующего элемента массива
Задача: Для каждого элемента массива создать новый массив, где каждый элемент — сумма текущего и следующего элемента исходного массива. Последний элемент оставить без изменений.


// пример: [1, 2, 3, 4] -> [3, 5, 7, 4]


Задание 10. Проверить, является ли массив строго возрастающей последовательностью
Задача: Вернуть true, если каждый следующий элемент больше предыдущего, иначе — false.

// пример: [1, 2, 3, 4] -> true
// пример: [1, 2, 2, 3] -> false
